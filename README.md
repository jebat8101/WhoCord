## Installation

1. Clone the repository (or download the ZIP) and open a terminal inside the folder.

2. Install the Python package and its dependencies:
   ```bash
   pip install -e .
   ```

3. Install the external command‑line tools:
   ```bash
   pip install sherlock-project maigret holehe h8mail gitfive naminter linkook sociopath
   ```

4. Run the tool:
   ```bash
   discord-osint
   ```

   The interactive menu will guide you from there.
